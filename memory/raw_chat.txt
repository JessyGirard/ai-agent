I am building a personal AI agent called ai-agent that is designed to grow with me over time.

This is not a simple chatbot. It is a long-term system that will integrate tools, memory, reasoning, and personal awareness.

Right now I am working on Memory System V2.

The goal of this system is to extract meaningful long-term memory from conversations instead of storing raw logs.

I want memory that understands meaning, filters noise, and evolves over time.

The system should avoid storing temporary emotions, greetings, or useless filler content.

It should focus on identity, goals, preferences, and active projects.

I prefer to learn step by step and validate each step before moving forward.

I do not like when too many things are explained at once or when instructions are unclear.

I am building this system in a modular way using folders and files where each part has a clear responsibility.

I understand that folders represent architecture and files represent components.

I already built a basic agent that runs in the terminal using python playground.py.

I also added tool support including a fetch_page tool that allows the agent to access the web.

I successfully connected my project to GitHub and learned how to push safely without exposing API keys.

Right now I want to improve the memory system so it becomes high quality and not noisy.

The memory should merge repeated ideas, track confidence, and evolve based on evidence.

I want the system to remain simple, controllable, and efficient.

I do not want unnecessary complexity or bloated architecture.

This project is part of a bigger goal which is to build a powerful AI super agent that can assist me in my work and learning.

This agent should eventually understand me, remember important things, and adapt to my workflow.

I am currently in the learning and building phase and I am actively improving the system step by step.

I care about doing things properly, even if it takes more time.
